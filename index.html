What is HTML?

HTML (HyperText Markup Language) is the skeleton of the web. No muscles, no skin! Just bare bones telling the browser what goes where. Before CSS makes it pretty and JavaScript makes it dance, HTML is there doing the unglamorous work of holding everything together. HTML uses elements to describe the structure and meaning of content. The browser reads your HTML and renders it as a visual page.

Basic HTML structure

Every HTML file starts with <!DOCTYPE html> a one-liner that tells the browser to use modern HTML5 rules. It's like a letter. The <head> is the envelope; nobody reads it but it tells the post office what to do. The <body> is the letter itself. 


<!-- A minimal HTML5 document -->
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Common HTML elements

HTML has over 100 elements, but you will use a small set most of the time: 


<!-- Headings: h1 is largest, h6 is smallest -->
<h1>Main heading</h1>
<h2>Sub-heading</h2>

<!-- Paragraph -->
<p>This is a paragraph of text.</p>

<!-- Hyperlink -->
<a href="quiz.html">Go to quiz</a>

<!-- Image -->
<img src="assets/img/photo.jpg" alt="My photo" />

<!-- Unordered list -->
<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

Semantic HTML

Semantic elements describe the purpose of the content, not just how it looks. This helps search engines and screen readers understand your page. 


<header>  <!-- top of the page -->
<nav>     <!-- navigation links -->
<main>    <!-- main page content -->
<section> <!-- a standalone section -->
<article> <!-- self-contained content -->
<footer>  <!-- bottom of the page -->

What is CSS?

CSS (Cascading Style Sheets) controls how HTML elements look. While HTML defines the structure of a page, CSS defines its appearance — colours, fonts, spacing, and layout.

CSS syntax


/* selector { property: value; } */

h1 {
  color: #00BFFF;
  font-size: 2rem;
  text-align: center;
}

p {
  color: #8b95a8;
  line-height: 1.7;
  margin-bottom: 1rem;
}

Selectors


/* Element selector */
p { color: white; }

/* Class selector */
.highlight { background-color: yellow; }

/* ID selector */
#logo { width: 120px; }

/* Descendant selector */
nav a { text-decoration: none; }

The box model

Every HTML element is a rectangular box with content, padding, border, and margin. 


.box {
  width: 300px;
  padding: 16px;
  border: 2px solid #00BFFF;
  margin: 24px;
}

Responsive design with Bootstrap

Bootstrap provides a grid system using 12 columns.


<div class="row">
  <div class="col-md-6">Left column</div>
  <div class="col-md-6">Right column</div>
</div>

What is JavaScript?

JavaScript (JS) is a programming language that runs in the browser. While HTML provides structure and CSS provides style, JavaScript adds behaviour — responding to user actions, updating content dynamically, and communicating with servers.

Variables and data types


// Declaring variables
const siteName = "WeaselCode";  // string
let score = 0;                  // number
let passed = false;             // boolean

// Array
const topics = ["HTML", "CSS", "JavaScript"];

// Object
const user = {
  name: "Alex",
  score: 8,
  passed: true
};

Functions


function greet(name) {
  return "Hello, " + name + "!";
}

const message = greet("Alex");
console.log(message); // "Hello, Alex!"

DOM manipulation


const heading = document.getElementById("main-title");
heading.textContent = "New title!";
heading.style.color = "#00BFFF";

Events


const btn = document.getElementById("my-btn");

btn.addEventListener("click", function() {
  alert("Button was clicked!");
});
