# WeaselCodes 
An interactive website built for the CITS5505 Agile Web Development unit individual project. The site teaches the fundamentals of HTML, CSS, and JavaScript through a tutorial page, tests knowledge via a dynamic quiz, and presents the author's professional profile.

## Pages

| Page | File | Description |
|---|---|---|
| Tutorial | `index.html` | Explains HTML, CSS, and JavaScript via a three-tab system with code snippets, interactive demos, and embedded videos |
| Quiz | `quiz.html` | 10+ questions loaded via AJAX from a local JSON file, randomised on each load, with score tracking and localStorage history |
| AI Reflection | `reflection.html` | Critical evaluation of AI tool usage throughout the project |
| About Me | `cv.html` | Professional CV  |
